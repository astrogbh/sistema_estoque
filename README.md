### Stock Manager 
Sistema de Estoque de uma Loja de Materiais de Construção
Cadastre clientes, funcionários e produtos

#### Programas Necessários
- Python
- MySQL
- Streamlit

#### Passo a Passo
- Importar o banco de dados fornecido
- Abrir o CMD, selecionar a pasta do projeto
- Digitar "streamlit run app.py" para rodar o projeto
